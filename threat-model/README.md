# Threat Model Exports (OWASP Threat Dragon)

Esta carpeta contiene los modelos de amenazas exportados en formato JSON de OWASP Threat Dragon.

Archivos:
- `01_SecureVault_Operativo_Threat_Dragon.json`: modelo principal del sistema (DFD operativo con STRIDE).
- `02_SecureVault_CICD_Threat_Dragon.json`: modelo de cadena CI/CD y supply chain (DFD CI/CD con STRIDE).

Cobertura:
- DFD Nivel 0 y Nivel 1 referenciados en documentación técnica.
- Amenazas STRIDE por elementos y flujos.
- Evidencia lista para importación directa en OWASP Threat Dragon.
