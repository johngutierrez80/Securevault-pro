# Manual de Usuario

## Registro
1. Abrir frontend en http://localhost:5173.
2. Seleccionar Registro.
3. Ingresar correo y contrasena (minimo 8 caracteres).

## Login
1. Cambiar a Login.
2. Ingresar credenciales.
3. El sistema habilita el dashboard.

## Gestion de secretos
1. Completar nombre y valor del secreto.
2. Opcionalmente establecer fecha de expiracion.
3. Guardar secreto.
4. Usar Refrescar para recargar lista.
5. Eliminar secreto cuando ya no sea necesario.
